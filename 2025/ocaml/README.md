# Advent of Code 2025

## Project Structure

```txt
.
├── _build                  # build outputs
├── bin                     # program entry points
├── docs                    # documentation
├── lib                     # public modules
├── test                    # tests
├── .editorconfig
├── .envrc
├── .gitignore
├── .ocamlformat            # ocamlformat configuration
├── CODEOWNERS
├── CONTRIBUTING.md
├── LICENSE
├── README.md
├── Taskfile.yml            # task runner configuration similar to make
├── dune-project	        # dune project configuration
├── flake.lock
├── flake.nix               # nix flake configuration
├── advent_of_code_2025.install  # autogenerated install script
└── advent_of_code_2025.opam     # autogenerated project description
```
